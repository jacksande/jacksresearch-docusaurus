# Inspired Version

Joseph Smith commenced to make corrections to the King James Version starting in June 1830 and finished the task on July 2, 1833. Smith’s version has been known as the Inspired Version. The following sections show differences and the similarities between the King James Version and Joseph Smith’s Inspired Version. In addition, an accompanying Hebrew (Old Testament) or Greek (New Testament) text is added for reference and further study.
